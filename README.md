# Gallery Museum

This is a DOM manipulation project I built as a FreeCodeCamp exercise in the form of a gallery of image thumbnails that expand when clicked to show the actual image with description texts. I added certain features in order to reinforce my knowledge.

## Technologies
HTML
CSS
JavaScript

## What I practiced
- DOM manipulation
- Event Listeners
- Array and Object Methods
